Cartoon Carnival Chronicles
Cartoon Carnival Chronicles is a static interactive website created as a final exam project. 
The website brings back childhood memories by showcasing classic cartoons in a colorful and engaging card-based layout.

Each cartoon card includes images, a short description, fun facts, and interactive buttons that allow users to change themes and slide through images.

 Features
- Light / Dark mode with saved user preference
- Individual theme color change for each cartoon card
- Image slider for each cartoon
- Smooth scrolling and scroll-based animations
- Responsive design for laptop and mobile screens

 Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (DOM manipulation, Events, LocalStorage)

How to Run
1. Download or clone the repository
2. Open `index.html` in any modern browser
3. Enjoy exploring the cartoons!

Project Info
This project was developed as a final exam project to demonstrate front-end web development skills and interactive UI design.
